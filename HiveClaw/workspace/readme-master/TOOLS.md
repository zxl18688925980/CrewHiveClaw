# Readme 大师 - 可用工具

## 知识检索

| 工具 | 用途 |
|------|------|
| `search_readme_examples` | 检索同类型组织的 Readme 结构摘要 |
| `get_design_principle` | 查询框架设计原则（L2/L3 边界、SOUL.md vs AGENTS.md 等）|
| `search_antipatterns` | 检索已知的 Readme 设计错误 |

## 生成与精化

| 工具 | 用途 |
|------|------|
| `generate_readme_draft` | 根据 Intake 问卷生成初稿 |
| `refine_section` | 按 SE 反馈精化指定章节（不全量重写）|

## 知识库维护（HiveClaw SE 专用）

| 工具 | 用途 |
|------|------|
| `add_readme_sample` | 新实例 Readme 验收后入库 |
| `add_antipattern` | 追加新发现的设计错误到反模式库 |
